# OWASP LLM Top 10 (2026)

Use this as the current LLM threat taxonomy for GenAI and LLM-enabled systems.

How to use:
- Check how untrusted content reaches the model: direct input, retrieved content, tool output, MCP, memory, and multimodal input.
- Check how model output can affect downstream systems, especially tools and privileged actions.
- Combine with MITRE ATLAS for adversarial AI techniques and with OWASP LLM Apps 2.0 as supplemental grounding.
