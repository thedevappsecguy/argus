# AIUC-1 Agent Standard

Use this skill for agent security, safety, reliability, governance, and assurance controls.

How to use:
- Use AIUC-1 as an assurance lens for agentic systems, not as a vulnerability taxonomy.
- Check security, data/privacy, accountability, reliability, and safety control themes.
- Combine with OWASP Agent Security, OWASP LLM, MCP Top 10, and MITRE ATLAS.
