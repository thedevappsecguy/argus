# OWASP Top 10 MCP (2025)

Use this skill for Model Context Protocol servers, clients, tools, and tool metadata.

How to use:
- Inspect credentials, scopes, tool descriptions, command execution, auditability, and context sharing.
- Combine with agentic and LLM skills because MCP channels often re-enter model context.
- Cite MCP item IDs from `resources/list.yaml`.
