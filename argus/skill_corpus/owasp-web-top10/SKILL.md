# OWASP Top 10 (2025) - Web Application Security

Use this skill for browser-facing applications, web frontends, and web backends.

How to use:
- Check access control, injection, misconfiguration, auth/session, SSRF, crypto, logging, and dependency risks.
- Cite item IDs from `resources/list.yaml` when raising web threats.
- Pair with Cheat Sheets and ASVS for concrete controls.
