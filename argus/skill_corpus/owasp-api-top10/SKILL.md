# OWASP API Security Top 10 (2023)

Use this skill for API endpoints and API-to-API flows.

How to use:
- Check every endpoint for object, property, and function-level authorization.
- Consider resource exhaustion, SSRF, inventory gaps, and unsafe upstream API consumption.
- Cite item IDs from `resources/list.yaml` when raising API threats.

Source: https://github.com/OWASP/API-Security/tree/master/editions/2023/en
