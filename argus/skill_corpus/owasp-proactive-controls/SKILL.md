# OWASP Proactive Controls

Use this skill to propose secure-by-design controls before or alongside vulnerability-specific mitigations.

How to use:
- Prefer proactive controls for systemic mitigations that prevent a class of findings.
- Combine with ASVS for verification and Cheat Sheets for implementation.
- Cite control IDs from `resources/list.yaml`.

Source: https://github.com/OWASP/www-project-proactive-controls/tree/master/docs/the-top-10
