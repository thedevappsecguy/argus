# OWASP Agent Security - Agentic Top 10 Candidates

Use this skill for agentic systems where LLM output drives memory, tools, workflows, or other agents.

How to use:
- Apply the candidate risks to every agent, tool, memory store, and inter-agent channel.
- For agentic LLM systems, combine with OWASP LLM Top 10 2026, AIUC-1, MITRE ATLAS, and MCP Top 10 when applicable.
- Rule out a candidate only when the system model shows no relevant memory, tool, identity, human approval, or inter-agent path.
